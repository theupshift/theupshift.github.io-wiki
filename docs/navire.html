Inventory

Navire is a Victorian-era laptop with 2GB of RAM and a 1.10 GHz processor. I run Linux on it in order to keep resource usage as lightweight and minimal as possible. Dotfiles are available on GitHub.

I've been meaning to switch to Debian or Void Linux but it's not a priority.

Essentials

Purpose Tool
WM i3wm
Terminal xterm, st
Browser Firefox, Chromium, surf
Text editor vim
Music player cmus
Files nnn, pcmanfm
Image editor GIMP
Music composition Marabu
Vector tool Dotgrid
Media player mpv
RSS reader newsboat

Usage

  • No idling in front of the computer
  • Focus on one task at a time
  • Only go online if necessary

Future

When fate presents the required resources and opportunities, I'd like to upgrade this machine, either through the purchase of a more powerful laptop or simply the swapping of components for more powerful alternatives -- additional RAM, an SSD, etc.
