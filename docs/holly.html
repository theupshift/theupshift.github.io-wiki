Workshop

Holly is a computer console built with a Raspberry Pi Zero W, features a 2.7-inch monochrome OLED display, and is powered by a 2700mAh lithium ion battery. The project's goal is to build a portable Linux console that can be used for simple tasks like writing notes and lightweight coding on the go.

Status

The project is put on temporary stasis as I messed up the soldering of the battery circuit.
