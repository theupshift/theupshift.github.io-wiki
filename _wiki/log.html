Log is a log and time-tracking system inspired by similar systems by Deuveir°, Neauoire° and vi°. It is available for Linux, macOS, and Windows and can be downloaded here°.

Usage

Log is designed for single-tasking: start a log entry, focus on your task, and then stop.

Commands are entered via a console that appears on keypress. The app has a built-in Guide that contains a list of commands and how to use them.

Future

A CLI variant and a mobile app is being planned.
