Current state

I'm currently using the following tools and services. These are the ones I recommend:

  • Browser: Firefox
  • E-mail: ProtonMail
  • Social: Mastodon
  • Metadata remover: mat

These are services I'm using that I don't fully advocate for and may ditch in the near future:

  • Duolingo: Unfortunately uses Facebook SDK
  • GitHub: Acquired by Microsoft; still fine for now
  • Google: Slowly being purged from my life
  • Outlook: Eww. The university created Microsoft accounts for every student so I'm stuck with this until I graduate.
  • Twitter: Annoying but it's unfortunately my main means of keeping in touch with friends ever since ditching Facebook.
  • VSCO: It's a great alternative to Instagram. I don't think I've ever had or heard of privacy issues with VSCO so far.
