Attic

A personal wiki

Structure

The Ath (Athenaeum) is built with modified variants of Neauoire's Oscean wiki engine and Indental format. Pages are parsed and generated from flat-file databases based on five page types: home, portal, project, commonplace, and status. Portals contain indexes of pages under their category. Project pages contain information about a project, including summaries from my time-tracking data. Notes contain unstructured notes on various topics. The Status page details the wiki's status — completion of documentation, missing elements, &c.

Design

Inspired by Low-Tech Magazine's solar site, the Ath is designed to be simple with only the elements essential for a pure reading experience. There are no ads, trackers, analytics, or popups. The only JavaScript code that runs on this site are a couple of scripts that handle search and the 404 router.

CSS

There are stylesheets for each page type to, as much as possible, reduce unused styles in pages. As a result, the stylesheets are below 1 kilobyte.

Future

The Ath is currently hosted on GH Pages but I'd like to migrate hosting onto a solar-powered server.
