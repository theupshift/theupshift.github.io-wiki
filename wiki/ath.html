Attic

The Ath (Athenaeum) houses information about me and what I do. I hope it also inspire others in some way.

Structure

The Ath is built on top of stripped down variants of Neauoire's Oscean wiki engine and Indental format. Pages are parsed and generated from flat-file databases based on five page types: home, portal, project, commonplace, and status. Portals contain indexes of pages under their category. Project pages contain information about a project, including summaries from my time-tracking data. Commonplace pages contain unstructured notes. The Status page details the wiki's status — completion of documentation, missing elements, &c.

Design

Inspired by Low-Tech Magazine's solar site, the Ath is designed to be simple with only the elements essential for a pure reading experience. There are no ads, trackers, analytics, or popups. The only JavaScript code that runs on this site are a couple of scripts that handle search and the 404 router.

CSS

Rather than having one universal stylesheet, the Ath has four stylesheets for four page types. This is to, as much as possible, avoid unused styles in pages. As a result, the Ath's stylesheets are just 562–854 bytes.

Optimisations

I used some arguably unnecessary tricks to reduce page weight as much as possible. This included removing optional HTML tags.

Future

The Ath is currently hosted on GitHub Pages but I would like to someday host it on a solar-powered server.
