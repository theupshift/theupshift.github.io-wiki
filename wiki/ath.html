The Ath (Athenæum) is a personal wiki created to document my Oeuvre in the hopes of inspiring others.

Structure

The site is static and is built with modified variants of Neauoire's Oscean wiki engine and Indental format. Pages are generated from flat-file databases called tomes. There are four page types, including the Home page:

  • Portals are indexes of related pages.
  • Archetype pages contain project info or notes.
  • The Status page details the wiki's status.

The generator is built in NodeJS. I have a command set to automatically build and push changes to GitHub:

$ ath -b

Design

Inspired by Low-Tech Magazine's solar site, the site is designed to be as energy-efficient as possible.

A simple single-column layout restricts the site to only the elements essential for a pure reading experience.

There are stylesheets for each page type to optimise code coverage. Classes and IDs are reduced to single letters. Each stylesheet is less than 1kB.

The site is constantly being redesigned in my endless effort to reduce its carbon footprint in any way possible.

Scripts

This site has no ads, trackers, or analytics. No JavaScript code runs on the client side. Everything is pure HTML+CSS.

Hosting

The site is hosted on GH Pages and ZEIT Now.
