The Ath (short for Athenaeum) is a wiki that serves as a personal knowledge base as well as a place where people can come and learn more about the things I do.

Structure

The site is built on top of stripped down variants of Devine Lu Linvega's Oscean wiki engine and Indental format.

As of 13Y18, the wiki features a static site generator. The site is locally built and pushed to GitHub whenever changes are made.

Design

Taking inspiration from Low-Tech Magazine's solar site, the site is designed to be simple, featuring few elements essential for a pure reading experience. Images are desaturated and dithered. The typeface is set to your browser's default. While it isn't a requirement, I try to keep each page below 100 kB (including assets). This keeps the website light and fast.

There are no ads, tracking scripts, and popups. The only JavaScript code that runs on this site are a couple of scripts for the search functionality and the 404 page handler. Since everything is static, viewing this site with JS disabled won't be much of a hassle.

Templates

There are four page templates — home, page, portal, and status. Pages are either project or commonplace pages. Portals contain indexes of links to pages. The Status page contains information on the status of the wiki's pages and a summary of my time-tracking data.
