Inventory

Navire is a Victorian-era Linux laptop with 2GB of RAM and a 1.10 GHz processor. Dotfiles are available on GitHub.

Essentials

  • WM - i3wm
  • Terminal - xterm, st
  • Browser - Firefox, surf
  • Text editor - vim
  • Music player - cmus
  • Files - ranger
  • Image editor - GIMP
  • Music composition - Marabu
  • Vector tool - Dotgrid
  • Media player - mpv
  • RSS - newsboat

Usage

  • No idling in front of the computer
  • Focus on one task at a time
  • Only go online if necessary
