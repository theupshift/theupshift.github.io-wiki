Inventory

My primary workstation is a Linux laptop from the Victorian era with 2GB of RAM and a 1.1GHz processor.

Usage

  • No idling in front of the computer
  • Focus on one task at a time
  • Work offline
  • Go online only when necessary

Essentials

  • Browser: Firefox
  • Files: ranger, nnn
  • Image compressor: optipng, jpegoptim
  • Image editor: GIMP
  • Image viewer: feh
  • Media player: mpv
  • Metadata cleaner: mat
  • Music composition: Marabu, MuseScore
  • Music player: cmus
  • PDF reader: zathura
  • Terminal: xterm, st
  • Text editor: vim
  • Vector tool: Dotgrid
  • WM: i3wm

Dotfiles

I'm constantly making the most of this machine by regularly optimising workflows and trying out (or building) lighter software alternatives to reduce resource usage. Dotfiles are available on GitHub.
