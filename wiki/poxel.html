Attic

Poxel is a pixel art editor written in 864 bytes of JS. View it here.

Features

To keep Poxel below 1kB, I could only add a few features: a resizable canvas, a colour picker, session saving, and the option to save as PNG.

The square canvas can be resized via a URL hash containing two values separated by a dash. The first value indicates the canvas size in pixels; the second indicates the size of the grid cells relative to the canvas size. For example, if you wanted a 512-pixel canvas with 32 rows and columns:

/poxel#512-32

The session is saved in the browser's localStorage. It persists until you either clear or resize the canvas.
